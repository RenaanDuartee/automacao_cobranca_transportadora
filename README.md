# 📦 Automação de Cobrança de Pendências Logísticas

## 🚀 Descrição

Script desenvolvido em Google Apps Script para automatizar o envio de e-mails de cobrança de pendências (DP e DT) para transportadoras.

## 🧠 Problema

O acompanhamento manual de pendências logísticas gera atrasos e demanda muito tempo.

## 💡 Solução

Automação integrada com Google Sheets que através da atualização das pendências é enviado e-mails de forma massiva apenas selecionando as transportadoras.

## 🛠 Tecnologias

* Google Apps Script
* Google Sheets
* Gmail

## ▶️ Funcionamento

1. Lê dados da planilha
2. Agrupa por transportadora
3. Envia e-mails

## 📈 Resultado

Melhoria no controle e redução de tempo operacional.
